# Assets Folder

This folder contains static assets for the application:

- **logos/** - Application logos (SVG, PNG, etc.)
- **favicons/** - Favicon files (favicon.ico, apple-touch-icon.png, etc.)
- **images/** - Other images used in the application

## Usage

Assets in this folder can be referenced directly in HTML or via public URLs:

```html
<!-- In index.html -->
<link rel="icon" href="/assets/favicons/favicon.ico" />

<!-- In React components -->
<img src="/assets/logos/logo.svg" alt="MyRevuHQ" />
```
