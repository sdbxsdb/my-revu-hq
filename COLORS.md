# Color Palette

## Primary Colors

### Dark Background Colors
- **Main Background**: `#030712` (Tailwind `gray-950`)
- **Card/Modal Background**: `#1a1a1a`
- **Darker Card Background**: `#141414`
- **Border/Divider**: `#2a2a2a`
- **Hover State**: `#333333`

### Turquoise/Teal Colors
- **Primary Teal (Links, Accents)**: `#2dd4bf` (Tailwind `teal-400`)
- **Light Teal (Hover)**: `#5eead4` (Tailwind `teal-300`)
- **Dark Teal (Borders)**: `#0d9488` (Tailwind `teal-600`)

## Usage in Code

### Dark Colors
- `bg-gray-950` → `#030712` (main page background)
- `bg-[#1a1a1a]` → Modal/card backgrounds
- `bg-[#141414]` → Darker card backgrounds
- `border-[#2a2a2a]` → Borders and dividers

### Teal Colors
- `text-teal-400` → `#2dd4bf` (primary links and text)
- `text-teal-300` → `#5eead4` (hover states)
- `border-teal-600/30` → `#0d9488` with 30% opacity (borders)
- `color="teal"` → Mantine component color (uses `teal-400`)

## Text Colors
- **Primary Text**: `#f3f4f6` (Tailwind `gray-100`)
- **Secondary Text**: `#d1d5db` (Tailwind `gray-300`)
- **Muted Text**: `#9ca3af` (Tailwind `gray-400`)
